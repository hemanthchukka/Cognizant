# Exercise 02 - Scrum Framework

## Objective
Learn and practice Scrum methodology.

## Topics Covered
- Scrum roles (Scrum Master, Product Owner, Team)
- Scrum ceremonies (Sprint planning, standup, review, retrospective)
- User stories and acceptance criteria
- Sprint execution

## Tasks
1. Understand Scrum roles and responsibilities
2. Plan a sprint
3. Create user stories
4. Conduct sprint ceremonies
